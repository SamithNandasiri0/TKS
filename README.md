# TKS
## SLTF Scoring Wi-Fi
Scoring system for teakwondo

## How to clone
1. Clone the repository `git clone https://github.com/SamithNandasiri0/TKS.git` or download the zip file from the repository and extract it
2. Then `cd TKS`
3. Run `npm install` for install dependencies

## How to use
1. Run `npm start`
2. Open `http://localhost:3000/scoreboard` for scoreboard
3. Open `http://localhost:3000/admin` for admin
4. Open `http://localhost:3000/judge` or scan the QR code on the admin page for judge

## How to connect to Wi-Fi
1. Connect to the Wi-Fi network named "SLTF Scoring Wi-Fi"
2. Open `http://localhost:3000/scoreboard` for scoreboard
3. Open `http://localhost:3000/admin` for admin
4. Open `http://localhost:3000/judge` or scan the QR code on the admin page for judge
