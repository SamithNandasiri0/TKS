# TKS
## SLTF Scoring Wi-Fi
Scoring system for teakwondo

## How to clone
Open the CMD and create an empty folder first. `mkdir <FOLDER_NAME>`
Then go to that folder. `cd <FOLDER_NAME>`
1. Clone the repository `git clone https://github.com/SamithNandasiri0/TKS.git` or download the zip file from the repository and extract it
2. Then `cd TKS`
3. Run `npm install` for install dependencies

## How to use
1. Run `npm start`
2. Open `http://[IP]:[port]/scoreboard` for scoreboard*
3. Open `http://[IP]:[port]/admin` for admin*
4. Open `http://[IP]:[port]/judge` or scan the QR code on the admin page for judge*

## How to connect to Wi-Fi
1. All devices should be connected to the same Wi-Fi network
2. Open `http://[IP]:[port]/scoreboard` for scoreboard*
3. Open `http://[IP]:[port]/admin` for admin*
4. Open `http://[IP]:[port]/judge` or scan the QR code on the admin page for judge*

* *The IP and the port will be shown in the terminal after running `npm start`. Default port is `4444`
